SimpleTalk

A simple chat application that can be easily integrated into bigger systems. This application uses:

  • Jquery (latest release available)
  • javascript-linkify from Cowboy
  • Smileys from Wikimedia commons Wikimedia

Features

  • Simple for users to join and talk.
  • Cross-plattform
  • Easy on server resources
  • Stable
  • Easy accessible logs
